What is the difference between var and let?

You cannot redeclare using the let keyword but you can reassign and redeclare with var. Let create block scope while with var you can access a hoisted variable.

What is the difference between var and const?

Const creates block scope. You cannot redeclare or reassign using the const keyword but you can reassign and redeclare with var. 

What is the difference between let and const?

You cannot redeclare or reassign using the const keyword but you can reassign with let.
